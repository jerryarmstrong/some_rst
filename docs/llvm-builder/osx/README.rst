    ## Solana Customized LLVM MacOS

Builds natively on MacOS

### Dependencies

* clang by Xcode6 or later
* cmake
* ninja

To install cmake and ninja you can use https://brew.sh

```bash
brew update
brew install cmake
brew install ninja
```
