“””Base class for all derived config classes.

This class provides some convenient functionality:

  • Disallows extra fields when constructing an object. User error should be reduced by exact arguments.

  • “one_of” fields. A subclass can group optional fields and enforce that only one of the fields be set. For example:

    ``` class ExampleConfig(BaseConfig):

    x: int = Field(None, one_of=”group_1”) y: int = Field(None, one_of=”group_1”)

    ExampleConfig(x=1) # ok ExampleConfig(y=1) # ok ExampleConfig(x=1, y=1) # throws error ```
