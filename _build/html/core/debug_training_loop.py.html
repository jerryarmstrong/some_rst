It is not intended for actual model tranining (it is not fast, doesn’t compile the model). It does not support checkpointing.

) -> None:

logging.warning(“Running debug training loop, don’t use for model training.”)

data_iter = iter(dataset) for step in range(0, train_steps + 1):
